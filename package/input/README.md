# OmiInput

## 安装

```js
import { Input } from 'omiui';
//or
import { Input } from 'omiui';
Vue.use(Input);
```

## Props

| name  | type          | default | description |
| ----- | ------------- | ------- | ----------- |
| type  |               | text    |             |
| value | String Number |         |             |

## Events

| name   | params | description |
| ------ | ------ | ----------- |
| input  | value  | 输入值      |
| search | Event  | -           |
| focus  | Event  | -           |
| blur   | Event  | -           |
